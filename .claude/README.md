# Claude Code Configuration Directory

This directory contains configuration and context files specifically for Claude Code AI assistant.

## Files in this Directory

- **prompts.md**: Reusable prompt templates for common development tasks
- **project-context.md**: Additional project context and workflows for Claude
- **README.md**: This file

## Files in Root Directory

- **`.clauderc`**: Main Claude Code configuration file
- **`.claudeignore`**: Files to exclude from Claude's context

## How to Use

Reference these files when working with Claude:
- "Use the Security Analysis prompt"
- "Review the project context"
- "Follow the deployment checklist"

These files help Claude provide more context-aware assistance for this specific project.
